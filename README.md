## Django 
1. Create A File
2. Create A Virtual Env 
```sh
virtualenv (V ENV Name) 
cd (V ENV Name)
source bin /activate
```
3. Install Django Using Pip3 
```sh
sudo pip3 install django
```
4. Create A Django Project 
```sh
django -admin startproject project name 
cd project name 
```
5. Run Python Server 
```sh
python manage.py runserver
```
